# SmartParser
Умный парсер маркетплейсов | Отчёты + Аналитика для бизнеса и частных покупок

> ❗ Тестовая версия проекта работает только на books.toscrape.com

Экономьте часы на поиске товаров — парсите несколько площадок одновременно, \
фильтруйте по множеству параметров, стройте диаграммы закупок и выгружайте отчёты в CSV.

### Подходит для:
+ 🔍 обычных покупателей (сравнение цен/рейтингов)
+ 👨‍💼 закупщиков, малого бизнеса и финдиректоров (аналитика + интеграция с сервисом учёта покупок)

## ✨ Возможности, которые вас удивят!

### Для всех пользователей:
+ 🔍 Двойной поиск — одновременно несколько магазинов
+ 🔀 Гибкие фильтры:
  + диапазон цены
  + рейтинг товара
  + сроки доставки
+ ⚖️ Сравнение одинаковых товаров на нескольких площадках в одной таблице

### Для бизнеса (B2B / команды):
+ 📊 Встроенная аналитика
+ 🧾 Собственный сервис учёта покупок:
  + добавляйте данные вручную или напрямую из парсера
  +храните историю закупок по проектам/отделам
+ 📑 Готовые отчёты для бухгалтерии
+ 💎 Расширенные функции — по подписке или фиксированная плата за один B2B-отчёт.

### 📦 Установка (скачайте с GitHub)

Клонируйте репозиторий \
```
git clone https://github.com/nemihail/smartparser.git
```
Установите зависимости
```
pip install -r requirements.txt
```

Запустите парсер
```
python main.py
```
или отдельно составитель отчетов:
```
python reportmaker.py
```

> ✅ Работает на Windows / macOS / Linux. ❗ Для парсинга нужен стабильный интернет
